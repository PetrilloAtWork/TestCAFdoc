Detailed Description

Creates a PMTnoiseGenerator algorithm object.

Implementations of this tool create an algorithm to emit PMT readout electronics noise, as an object derived from icarus::opdet::PMTnoiseGenerator<util::quantities::counts_f> (the template parameter is the same used by the single photoelectron response, but it's not programmatically tied to it to avoid entangling dependencies).

The tool object is just a wrapper factory that creates the actual generator, which is owned and can be returned for use.

The signature of the tool constructor should be compatible with:

PMTnoiseGeneratorTool(fhicl::ParameterSet const&)
