The ICARUS detector simulation and reconstruction software: icaruscode reference manual

This is the documentation of the ICARUS detector simulation and reconstruction software, distributed in the main GIT repository icaruscode.

This software is powered by the LArSoft toolkit, which has its own documentation in Doxygen format. That documentation is linked by the ICARUS documentation.

The documentation is automatically generated out of comments encoded in the source files in Doxygen format.

Sources for ICARUS documentation

ICARUS documentation is scattered in places:

  • wiki in GitHub (old version in Fermilab Redmine) offers guidelines, documents procedures, and much more; it tends to go outdated though;
  • SBN DocDB (https://sbn-docdb.fnal.gov/cgi-bin/sso/DocumentDatabase) contains plenty of information
  • software reference manual (this one; all versions: here; latest: here) is a reference guide for the use of specific ICARUS code objects.
  • ICARUS at work page is a hub for documentation on the detector and working groups
  • the main repository of the software is now in GitHub as icaruscode repository under the SBNSoftware umbrella; the SBN software documentation repository is also in GitHub
