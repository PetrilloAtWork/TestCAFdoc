Detailed Description

template<typename ADCT>
class icarus::opdet::ConstantPedestalGeneratorAlg< ADCT >

Constant pedestal generator algorithm for PMT electronics simulation.

Parameters
ADCTtype of the ADC count this interface treats with.

This generator produces a constant pedestal, with the addition of the noise from the configured noise generator.

Adds pedestal (with noise) to n samples starting at begin.

Parameters
channelID of the readout channel the samples are from
timethe absolute time of the first sample being filled [UTC, ns]
beginpointer to the first sample to be added with pedestal
nnumber of samples to add pedestal to
Returns
the number of samples actually added pedestal

No check is performed on the validity of the destination buffer. It is guaranteed that no more than n samples are changed, in the contiguous sequence after begin.

The return value should be n unless the generator was unable to fully fulfill the request.

The pedestal is added before the noise is added (that may be relevant if the noise level depends on the value of the sample).

Overwrites n samples starting at begin with pedestal plus noise.

Parameters
channelID of the readout channel the samples are from
timethe absolute time of the first sample being filled [UTC, ns]
beginpointer to the first sample to be overwritten with pedestal
nnumber of samples to fill with pedestal
Returns
the number of samples actually overwritten with pedestal

No check is performed on the validity of the destination buffer. Old values are typically completely overwritten. It is guaranteed that no more than n samples are overwritten, in the contiguous sequence after begin.

The return value should be n unless the generator was unable to fully fulfill the request.

The pedestal is set before the noise is added (that may be relevant if the noise level depends on the value of the sample).
