Detailed Description

Extracts a configured time from reconstructed optical hits.

This class provides three functionalities:

  • Type: an enumerator listing all the optical hit times supported by the extractor;
  • timeOf() (also operator()): returns the type of time configured in the object;
  • timeOf() (static version): returns the type of time specified as argument.

Note that the exact definition of each of the times depends on the algorithm used for the reconstruction of optical hits.

Example of usage:

recob::OpHitTimeSelector const hitTime { recob::OpHitTimeSelector::Start };
for (recob::OpHit const& opHit: OpHits) {
double const time = hitTime(opHit);
// ... do something with it
}
