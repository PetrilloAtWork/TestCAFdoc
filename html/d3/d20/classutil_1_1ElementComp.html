Detailed Description

template<typename Key, std::size_t N, typename Comp>
class util::ElementComp< Key, N, Comp >

Functor applying a specified function on a certain tuple element.

Template Parameters
Keytype of the key comparison
Nthe number of the element to apply the functor on
Compthe type of the functor

The Key type is the type of the element N of the tuples being compared.

Example of usage:

std::vector<std::pair<int, std::string>> names;
std::sort(names.begin(), names.end()); // sorts by int, then std::string
int code = 7;
if (!std::binary_search(names.begin(), names.end(), code, ElementComp<int, 0>{}))
std::cerr << "code=" << code << " not available" << std::endl;
