Detailed Description

Module which does nothing.

See also
DummyFilter

This module accepts any FHiCL configuration (which it ignores), acts like a producer that does not consume anything and does not produce anything.

It is a "good" way to remove a producer module from a job without having to mess with much configuration: just target it with a: 

physics.producers.redundantfilter.module_type: DummyProducer

Configuration parameters

Anything.
