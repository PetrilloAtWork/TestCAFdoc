ResourcePtr_t * anonymous_namespace{ChannelMapSQLite.cxx}::SQLiteResource< PTR >::reference ( )
inline

Returns the pointer to the address of the resource.

Note
The value pointed to is the actual address that this object tracks; this allows using this object with functions which set the address, but also provides a route to circumvent RAII, since the resource pointed by the overwritten value won't be freed.
