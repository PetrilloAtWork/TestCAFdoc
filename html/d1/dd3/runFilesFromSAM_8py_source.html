12__doc__ = f"""Create file lists from ICARUS data runs.
13
14For each query or definition, a SAM query is executed and the file locations
15are written all on screen or each into their own input file, depending on the
16arguments.
17Only one output pattern can be specified. That pattern can hold placeholders:
18 %n for the name as specified (may contain unacceptable characters!);
19 %s for a sanitized version of the name;
20 %t for a tag suitable for a file name;
21 %m for the requested maximum limit of entries;
22 %# for the number of input on the command line.
23
24If the automatic naming option is specified instead (equivalent to specifying
25an empty output pattern), a name is chosen with suffix '.filelist', that
26includes the verbatim definition or a sanitized version of the query, and the
27maximum number of entries requested, if any.
28
29If the output name ends with a directory separator ({os.sep}), it is used as the
30output directory, while the name is generated as if the output option were empty
31(that is, as the automatic naming were requested).
32
33By default, existing file lists are not overwritten.
34
35
36"""
