Detailed Description

template<typename ADCT>
class icarus::opdet::NoNoiseGeneratorAlg< ADCT >

No-noise generator algorithm for PMT electronics simulation.

Parameters
ADCTtype of the ADC count this interface treats with.

This noise generator always emits no noise (0).

The CLHEP random engine passed in the configuration is ignored.

Adds noise to n samples starting at begin.

Parameters

beginpointer to the first sample to be added with noise
nnumber of samples to add noise to
Returns
the number of samples actually added noise

No check is performed on the validity of the destination buffer. It is guaranteed that no more than n samples are changed, in the contiguous sequence after begin.

The return value should be n unless the generator was unable to fully fulfill the request.

Parameters
