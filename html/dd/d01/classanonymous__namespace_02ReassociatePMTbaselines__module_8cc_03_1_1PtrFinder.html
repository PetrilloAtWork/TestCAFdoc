Detailed Description

template<typename T, typename Comp = std::less<T>>
class anonymous_namespace{ReassociatePMTbaselines_module.cc}::PtrFinder< T, Comp >

Keeps a map of the art pointers of some objects.

Template Parameters
Ttype of the tracked objects; required to be comparable
Comp(default: std::less<T>) type of functor comparison

The object looks up a art pointer for an object which compares equal to a requested one.

To create an PtrFinder object, use makePtrFinder() helper function.

Requirements

Object of type Comp must support:

  • bool operator() (const T& a, const T& b) returning whether a is strictly before b (for whatever definition). This is a strict order comparison.
