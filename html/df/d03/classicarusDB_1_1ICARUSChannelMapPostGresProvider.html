Detailed Description

Interface to the PostgreSQL ICARUS channel mapping database.

The database is normally deployed on the network.

The implementation is fully delegated to icarusDB::ICARUSChannelMapProviderBase.

Configuration parameters

See icarusDB::ICARUSChannelMapProviderBase, except for:

  • ChannelMappingTool (algorithm configuration): see icarusDB::ChannelMapPostGres configuration.
