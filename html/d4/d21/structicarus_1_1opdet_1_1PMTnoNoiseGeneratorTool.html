Detailed Description

Creates a icarus::opdet::NoNoiseGeneratorAlg algorithm.

See also
icarus::opdet::NoNoiseGeneratorAlg

This tool creates a icarus::opdet::NoNoiseGeneratorAlg<counts_f> "noise" generator algorithm.

The usage pattern of this algorithm is the standard one documented in icarus::opdet::PMTnoiseGeneratorToolBase.

Configuration

Run lar --print-description PMTgausNoiseGeneratorTool (or read icarus::opdet::NoNoiseGeneratorAlg::Config data structure) for a short explanation of the meaning of the parameters, or check the documentation of icarus::opdet::NoNoiseGeneratorAlg.
