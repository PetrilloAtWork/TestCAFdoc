Detailed Description

template<typename ADCT>
class icarus::opdet::GaussianNoiseGeneratorAlg< ADCT >

Gaussian noise generator algorithm for PMT electronics simulation.

Parameters
ADCTtype of the ADC count this interface treats with.

This noise generator produces a random Gaussian-distributed noise with the specified RMS and no bias.

It requires a CLHEP random engine as input, and directly uses CLHEP::RandGaussQ adaptor on it.

Generating for any type ADCT other that the CLHEP-native double requires a conversion and slows down the generation.
